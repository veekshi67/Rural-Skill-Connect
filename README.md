# 🌾 Rural Skill Connect

## 📌 Overview
Rural Skill Connect is an AI-powered platform that connects skilled rural workers with nearby customers. It helps workers find local employment opportunities while allowing customers to hire trusted service providers quickly and easily.

---

## 🚀 Features

- Worker Registration
- Customer Registration
- Skill-Based Worker Profiles
- Search Nearby Workers
- Job Posting
- User Profiles
- Simple and Responsive Interface

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 🤖 AI Features (Lemma SDK)

- AI Worker Recommendation
- AI Job Description Generator
- AI Profile Enhancement
- Multilingual Assistance

---

## 🎯 Problem Solved

Many skilled rural workers lack digital visibility and struggle to find nearby job opportunities. Rural Skill Connect bridges this gap by connecting workers and customers through a simple digital platform.
